# Program Overview
This is a turtle library base program that generates diffrents pattern of shapes divided into 9 patterns.
The program randomly generates location , pen size , colors , and patterns for diffrent choices. 
## Shapes :
- Triangle shape
- Square shape
- Pentagon shape
## Patterns :
- One shape random
- Three shape random
- One sub shape random
- Three sub shape random
- Three sub shape and Three shape random
  
## Running the program :
Constanlty test shape by inputting 1 - 9 choices then close and reopen constantly
